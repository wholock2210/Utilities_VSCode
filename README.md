# Wholock Utilities

Gồm nhiều tiện ích khác nhau

## Tính năng

- Tự động chuyển chuỗi dài sang nhiều dòng khi nhấn Enter bên trong chuỗi C#.
- Nhấn Tab để thoát ra ngoài chuỗi nhanh chóng.
- Nhấn ; để kết thúc nhanh một dòng code
- bôi đen một dòng code bất kỳ và chuột phải sẽ có menu tiện ích 
    + utility synthesis
        * convert all letters to uppercase : nếu trong code được bôi đen có ít nhất 1 đoạn string thì sẽ chuyển tất cả chúng    
        thành chuỗi in hoa.
        * convert all letters to lowercase : nếu trong code được bôi đen có ít nhất 1 đoạn string thì sẽ chuyển tất cả chúng    
        thành chuỗi in thường.

## Kích hoạt

Extension sẽ tự động hoạt động khi mở file `.cs`.
